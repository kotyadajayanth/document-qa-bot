# Document Q&A Bot using RAG

A Retrieval-Augmented Generation (RAG) based document question-answering system built using Python, ChromaDB, Sentence Transformers, and Google Gemini.

## Features

- PDF document ingestion
- Text chunking
- Embeddings using all-MiniLM-L6-v2
- ChromaDB vector database
- Similarity search retrieval
- Gemini answer generation
- Source citations
- Streamlit UI

## Tech Stack

- Python
- ChromaDB
- Sentence Transformers
- Google Gemini
- Streamlit
